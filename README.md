# Project Title

An online banking API project using NodeJS and Mysql


# Getting started
- Clone the repository
```
git clone https://github.com/palhrz/bankify.git
```
- Install dependencies
```
cd bankify
npm install
```
- Add .env file
```
PORT=
JWT=
TOKEN_KEY=
```
- Build and run the project
```
node index.js
```
  Navigate to `http://localhost:8001`
